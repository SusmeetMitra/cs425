Welcome

This web application demonstrates the relational schema and ER design for a Real Estate Management system.

  • Register renters (Users + Renter tables).
  • Browse/search properties joined with Neighbourhood and property subtypes.
  • Create bookings using credit cards, with automatic reward points.
  • Renter dashboard showing bookings and total rewards.

For the demo, you can:

  1. Register a new renter.
  2. Browse properties and pick one.
  3. Create a booking.
  4. Open the dashboard at /renters/<renter_email>.
